# Claude Plugins

Personal collection of Claude Code plugins.

## Plugins

- **sidekick** - Neovim integration for Claude Code

## Commands

- **critic** - Create user personas to test and vet your work. Simulate realistic users reviewing your content, flows, or code.

## Usage

Add this marketplace to your Claude Code configuration to access these plugins.

## Plugin Sources

All plugins are maintained in their respective repositories. See `.claude-plugin/marketplace.json` for details.
